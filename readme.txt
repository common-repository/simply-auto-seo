Adds SEO tags to your website automatically without cluttering up the editor with input boxes you don't use.

== Features ==
* Great for people who don't want to mess around with custom SEO settings.
* Doesn't clutter the WordPress editor with input boxes you don't ever use.
* Post/Page description can be edited using the Post/Page excerpt.
* Post/Page description can be trimmed using the admin options to exclude unwanted text (i.e. "Read More").
* Categories, Tags, Archives etc. description is set by its own description.
* v1.1.0 added "Simply Auto SEO Admin" role and "Simply Auto SEO Settings" capability.
* v1.2.0 added "Enable Page Excerpts" option.
* v1.2.1 updated Tested up to: 6.1.
